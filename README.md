# Museo Virtual de Monumentos Emblemáticos.

Bienvenido al **Museo Virtual de Monumentos Emblemáticos**, un proyecto desarrollado en **Unity 6.1** que ofrece una experiencia interactiva en 3D para explorar algunos de los monumentos más representativos alrededor del mundo.  

## Objetivo
El propósito de este proyecto es brindar una experiencia educativa y cultural accesible, permitiendo a los usuarios recorrer un museo virtual y conocer monumentos emblemáticos mediante un entorno inmersivo y dinámico.

## Tecnologías utilizadas
- **Unity 6.1** para el desarrollo del entorno 3D  
- **C#** para la lógica de interacción  
- **GitHub Pages** para la publicación en la web  
- **Android APK** para acceso móvil  

## Acceso al proyecto
- **Repositorio en GitHub:** [MuseoVirtual](https://github.com/viviananinoreyes4-web/MuseoVirtual.git)  
- **Sitio activo en GitHub Pages:** [Museo Virtual Online](https://viviananinoreyes4-web.github.io/MuseoVirtual/)  
- **APK funcional para Android:** [Descargar APK en Google Drive](https://drive.google.com/file/d/1u1206P3i7-Pbmvcyq6lqxWKbYC297ng0/view?usp=sharing)  

## Características principales
- Recorrido virtual en 3D por monumentos emblemáticos.  
- Interfaz intuitiva y accesible.  
- Compatible con dispositivos móviles Android mediante APK.  
- Experiencia educativa y cultural pensada para estudiantes y público general.  

## Instalación del APK en Android
1. Descarga el archivo APK desde el enlace de Google Drive.  
2. Copia el archivo a tu dispositivo Android.  
3. Activa la opción de instalar aplicaciones desde orígenes desconocidos en tu dispositivo.  
4. Instala y disfruta del museo virtual.
